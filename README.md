# C言語・Linuxカーネル等学習用リポジトリ
このリポジトリでは、自分がC言語やLinuxカーネルなどを基礎から学ぶために書いた自作のプログラムをまとめている。

## 学習目的
- C言語とLinuxカーネルの学習から低レイヤーの動作を理解する
- bashやpythonなどの実用的な言語についても学び、実践的に使えるようにする

## 開発環境
### OS等
- OS: Fedora Linux 44 (Xfce) 
- カーネル: Linux 6.19.10-300.fc44.x86_64
- アーキテクチャ: x86_64 

#### 以前の開発環境
~2026/5/20
- OS: Ubuntu 24.04.4 LTS
- カーネル: Linux 6.17.0-14-generic
- アーキテクチャ: x86_64 

### コンパイラ等
- GCC: 13.3.0
- Shell: GNU bash 5.2.21 
- Python: 3.12.3
